## Flujo de GIT
1. Actualizar cambios
```sh
git pull upstream master
```
2. Trabajar en local
3. Agregar mis cambios
```sh
git add .|File|Directory
```
4. Realizar commit
```sh
git commit -m "..."
```
5. Actualizar cambios
```sh
git pull upstream master
```
6.1  Subir cambios
```sh
git push origin nombre_rama
```
6.2 Hacer pull request
7. Resolver conflictos manuales
